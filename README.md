# OSSFileTool
在红队行动中利用OSS进行文件传输的小工具

+ 支持大文件上传

+ 支持文件下载

+ 使用OSS文件下载可有效规避杀软对文件下载等敏感操作的查杀

+ Win10-Win7 支持
